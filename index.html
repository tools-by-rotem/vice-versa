Vice Versa

Effortlessly convert selected text between Hebrew and English in your browser.

Add to Chrome

Features

  • One-click text conversion via right-click menu or Ctrl+Alt.
  • Automatic detection of Hebrew or English text.
  • No external dependencies or internet required.
  • Lightweight and privacy-focused.

How to Use

  1. Install the extension from the Chrome Web Store.
  2. Select any text in your browser.
  3. Press Ctrl+Alt or Right-click and choose "Convert text from Hebrew to English or vice versa".
  4. The selected text will be replaced with the converted version.

Frequently Asked Questions

Does Vice Versa work offline?

Yes, the extension works entirely offline, ensuring your data remains private.

What languages are supported?

Currently, Vice Versa supports text conversion between Hebrew and English.

Can I undo the text conversion?

Simply use the browser's undo shortcut (Ctrl+Z or Cmd+Z) to revert the changes.
